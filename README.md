🚀 LAUNCHPAD SUPER DOPE 🚀
Launchpad専用の「自動生成リズムゲーム」兼「DJサンプラーシステム」です。
YouTubeのURLを投げ込むだけで、AIが楽曲を解析して譜面を自動生成し、さらに曲の「一番おいしいボーカルフレーズ」を切り出して右端のボタンにサンプラーとして割り当てます。

空間のうねり、迫りくるノーツ、そして背景で透過再生されるミュージックビデオ。そのすべてが楽曲のBPM（心臓の鼓動）に完全シンクロする、圧倒的な没入感とVJプレイを体験してください。

✨ 主な機能 (Features)
YouTubeからの全自動解析パイプライン

YouTubeのURLを入力するだけで、動画と音源を自動ダウンロード。

librosa のHPSS（高調波打楽器音分離）を用いて「ボーカル・メロディ成分」と「ドラム・ベース成分」を分離し、両方から美味しいリズムを抽出してLaunchpad用の譜面を自動生成します。

完全BPM同期の「DOPE 3Dエンジン」＆ 背景MV再生

楽曲のBPM（テンポ）を自動検知し、3Dトンネルの進むスピードや画面の脈動（FOVの拡大縮小）、ねじれの速度を「拍（ビート）」に完全に同期させています。

OpenCV により、ダウンロードした公式ミュージックビデオを背景にうっすらと透過再生。動画がない場合はワイヤーフレームのサイケデリック空間が描画されます。

サビ・カッター（Auto Chorus-Chop Sampler）

楽曲を6つのブロックに分割し、それぞれのブロックから「最も高音域（ボーカルやシンセ）のエネルギーが高いフレーズ」を自動探索。

クリップノイズを防ぐフェードアウト処理を施した極上の「声ネタ」を6つ自動生成し、Launchpadの右端ボタンにマッピングします。

🛠 必要な環境 (Requirements)
ハードウェア: Novation Launchpad (MIDI対応モデル)

Python: 3.8以上推奨

必要なライブラリ:

Bash
pip install pygame mido python-rtmidi librosa numpy pandas yt-dlp soundfile scipy opencv-python
(※ YouTubeからの音声抽出に ffmpeg がシステムにインストールされている必要があります)

🎮 遊び方 (How to Play)
1. 楽曲の追加と解析
ターミナルで python launchpad_superdope_video.py を実行します。
メインメニューで [0] を選び、好きなYouTube動画のURLをペーストしてください。
約1分かけて「動画/音源のダウンロード」「譜面生成」「BPM測定」「サンプラー用クリップの抽出」をすべて行います。

2. コントロール（Launchpadの配置）
ゲームプレイ中は、Launchpadが以下のような「楽器」兼「コントローラー」になります。

11 〜 18 (下から2段目の横一列)

リズムゲームのメインパッドです。奥から迫ってくるノーツが手前の判定ラインに重なった瞬間に叩きます。

89, 79, 69, 59, 49, 39 (右端の丸ボタン上から6つ)

サンプラーパッドです。「サビ・カッター」が自動抽出した6種類のボーカル・フレーズがアサインされています（待機中はマゼンタ色に点灯）。

いつ叩いても音が鳴り、画面全体に強烈なグリッチ（揺れ）エフェクトが走ります。プレイ中に自由にカットアップ（連打）して遊んでください。

19, 29 (右端の丸ボタン下から2つ)

SFXパッドです。あらかじめディレクトリに horn.wav (Airhorn) や scratch.wav を入れておくと、DJプレイのような効果音を鳴らせます。
